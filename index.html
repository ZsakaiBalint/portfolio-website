Tech stack

(technologies I use daily)

    Symfony

    Symfony is a powerful and flexible open-source PHP framework designed for building complex web applications and APIs. It provides a robust foundation for developers to create scalable and maintainable web projects with a structured approach. Symfony is known for its high performance, modularity, and extensive ecosystem.

    Doctrine

    Doctrine is a powerful Object-Relational Mapping (ORM) library for PHP. It simplifies database interactions by mapping PHP objects to database tables, allowing developers to work with databases using PHP code instead of SQL. Doctrine is known for its flexibility, performance, and support for complex database relationships, making it an essential tool for building robust and maintainable web applications.

    Postgre SQL

    PostgreSQL is a powerful, open-source relational database management system known for its reliability, scalability, and extensive feature set. It supports advanced data types and performance optimization, making it a preferred choice for complex web applications and data-driven projects. PostgreSQL's robust architecture and active community contribute to its continuous improvement and widespread adoption.

    JavaScript (Node.js)

    JavaScript (Node.js) is a versatile runtime environment that allows developers to execute JavaScript on the server-side. It enables the creation of fast, scalable network applications and real-time services. Node.js leverages the power of JavaScript, its non-blocking I/O model, and a vast ecosystem of modules to build efficient, event-driven applications that can handle numerous simultaneous connections with high throughput.

    JWT

    JSON Web Token (JWT) is a compact, URL-safe means of representing claims to be transferred between two parties. It is commonly used for authentication and authorization in web applications, allowing secure and efficient transfer of user information. JWT's standardized structure and cryptographic security ensure data integrity and trustworthiness across different systems and platforms.

    Git

    Git is a distributed version control system designed to handle everything from small to very large projects with speed and efficiency. It allows multiple developers to collaborate on code, track changes, and manage project history. Git's branching and merging capabilities, combined with its robust ecosystem of tools, make it an indispensable tool for modern software development.

    GitHub

    GitHub is a web-based platform that provides hosting for Git repositories. It offers a suite of collaborative features such as bug tracking, feature requests, task management, and wikis for every project. GitHub's integration with Git enables seamless code collaboration and sharing, making it an essential tool for developers and open-source projects around the world.

    Docker

    Docker is an open platform for developing, shipping, and running applications in containers. It enables developers to package applications with all their dependencies into a standardized unit for software development. Docker containers ensure consistency across multiple development and production environments, streamlining the deployment process and improving resource efficiency.

    PHPUnit

    PHPUnit is a popular testing framework for PHP, designed to support test-driven development. It provides a wide range of assertions and tools to create, organize, and execute tests, ensuring code quality and reliability. PHPUnit's integration with continuous integration systems and its extensive documentation make it a valuable tool for maintaining robust and error-free PHP applications.
